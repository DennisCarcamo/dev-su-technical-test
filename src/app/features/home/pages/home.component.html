Proyecto: Aplicación de Gestión de Productos Financieros

F1. Listado de productos financieros

Se requiere una aplicación para visualizar los diferentes productos financieros ofrecidos por la Institución Banco. Los datos se obtendrán desde una API externa.

  • Visualización de todos los productos.

F2. Búsqueda de productos financieros

El usuario debe poder buscar productos mediante un campo de texto.

  • Se debe filtrar la lista de productos según el criterio de búsqueda.

F3. Cantidad de registros

La aplicación debe mostrar la cantidad de resultados mostrados en la lista y permitir seleccionar cuántos registros mostrar por página.

  • Select con valores: 5, 10 y 20 registros.
  • Mostrar el número total de resultados.
  • Agregar paginación simple si hay muchos registros.

F4. Agregar producto

Se debe permitir crear un nuevo producto financiero desde la interfaz.

  • Formulario de registro con todos los campos requeridos.
  • Validaciones de datos según reglas de negocio.

F5. Editar producto

Se debe permitir modificar los datos de un producto existente.

  • Formulario prellenado con la información actual del producto.
  • Validaciones de datos y actualización mediante API.

F6. Eliminar producto

Se debe permitir eliminar un producto de manera segura.

  • Confirmación antes de eliminar.
  • Actualización automática del listado tras la eliminación.
